Public opt-in path for reviewers

End users opt in on the public website at https://whatsitdoing.app/ during paid automotive consult intake. The website does not require a private login to view the intake flow.

  1. User opens https://whatsitdoing.app/.
  2. User taps Continue and answers who is working on the vehicle.
  3. User enters vehicle symptoms, vehicle details, and optional files/audio.
  4. User may enter a phone number and actively select the optional checkbox: Text me case updates.
  5. The checkbox is optional and should not be preselected.
  6. Near the phone field, the user sees that text updates are optional and that details are provided before payment, with links to Privacy and Terms.
  7. Before clicking Pay & submit, the page shows the full SMS disclosure.
